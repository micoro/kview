File content

Inside the file can be found variables to manage the following settings:

  • Custom buttons settings (CustButtSett)
  • Custom panels (CustomPanels)
  • Import file conversion settings (ImpFileConvSett)
  • Import from workspace conversion settings (ImpWSConvSett)
  • User defined extensions (UserDefExt)
  • Default X axis (defaultXAxis)

Custom button settings

It is a 3x2 cell array. Each row represent one of the three custom buttons panels. The first column is the path to the folder that contains the *.buttontable file; the second column is the name of the file (with extension).

Default X axis

It is a string cell array of two elements. The first one is the name of the default subsystem while the second one is the name of the default variable. The two strings in this cell array will determine what happens when the user left clicks or right clicks the "set default" button for the X axis.
